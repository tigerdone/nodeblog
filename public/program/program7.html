Intelligent Smoke Alarm System with Wireless Sensor Network using ZigBee

We design an intelligent Smoke Alarm System that uses ZigBee transmission technology to build a wireless network, uses Random Forest to identify smoke, and uses E-charts for data visualization. By combining the real-time dynamic changes of various environmental factors, compared to the traditional smoke alarm, the accuracy and controllability of the fire warning are increased, and the visualization of the data enables users to monitor the room environment more intuitively. At present, the collected environmental data can be classified into four status, i.e.normal air, water mist, kitchen cooking and fire smoke.
