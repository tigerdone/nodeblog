Object Recognition

To improve children's interest in learning a second language, we combine object recognition technology with word learning. We selected items that are common in life as a word library, such as toothbrush, mango, cookies, CD, and then designed an interactive way for each word to match its characteristics. We project the program interface onto the desktop, and enter the corresponding learning interface according to the identified items. Children can learn interactively by clicking, dragging, etc.
